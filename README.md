# Official Firmware for the Dankboard

## Based from QMK

### Current Version 11.8

Supports: 

- Dankboard
- Dankboard-Ortho
- Dankboard-M
- Dankboard-S
- Dankboard-S-Ortho
- Dankboard-XS
- Dankboard-XS-Ortho
- Dankpad
